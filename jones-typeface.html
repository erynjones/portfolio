Jones Typeface

Font Design

How the Typeface was Created

Jones is a display font where the counters are used to align words horizontally in an artistic manner. In doing so, it creates a unique visual of how the type is displayed, especially throughout certain letter combinations such as g, o, b, d, p, q, and a. Jones is a font that creates contrast between the thick to thin transition. It is similar to how you would write out letters using a scroll tip marker that has a ridge in it, which is how the font was developed. It is best shown at a large point size, since it is a sans serif display font.
